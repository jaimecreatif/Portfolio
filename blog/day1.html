📖 React Study Notes - Day 1

1️⃣ What is React?

React is a JavaScript UI library developed by Meta (formerly Facebook).

  • ✅ Component-based: Reusable and independent UI components.
  • ✅ Virtual DOM: Efficiently updates only the changed parts of the UI.
  • ✅ Declarative approach: Focuses on "what" the UI should look like.

2️⃣ Setting Up React Development Environment

Instead of using Create React App (CRA), we use Vite, which is much faster and more lightweight.

✅ Installation Guide

1️⃣ Check if Node.js is installed:
    node -v
If not installed:
    brew install node

2️⃣ Create a Vite project:
    npm create vite@latest my-react-app --template react
    cd my-react-app
    npm install
    npm run dev

✅ Open http://localhost:5173 to see the project running! 🚀

3️⃣ Understanding JSX

JSX allows JavaScript and HTML-like syntax to be mixed in React components.

✅ JSX Rules

  • JSX must have a single parent tag.
  • JavaScript expressions must be wrapped in { }.
  • Use className instead of class.
  • Self-closing tags must end with /.

4️⃣ Understanding `useState`

`useState` is a React Hook used to manage a component's state.


import { useState } from "react";

function App() {
    const [count, setCount] = useState(0);

    return (
        <div>
            <p>Current count: {count}</p>
            <button onClick={() => setCount(count + 1)}>+1 Increase</button>
        </div>
    );
}

📝 Assignment: Applying JSX

✅ Solution


const [message, setMessage] = useState("Shall we start studying?");

<button onClick={() => setMessage(prev => prev === "Shall we start studying?" ? "Studying hard!" : "Shall we start studying?")}>
    {message}
</button>

🎯 Day 1 Summary

  • ✅ React is a JavaScript UI library.
  • ✅ Vite is used for fast project setup.
  • ✅ JSX allows HTML-like syntax inside JavaScript.
  • ✅ `useState` manages state in React components.

📌 React Study - Question Notes

1️⃣ What is the DOM?

The Document Object Model (DOM) is a tree-structured representation of a webpage.

2️⃣ What is Vite?

Vite is a faster build tool for React development, providing instant updates via HMR.

3️⃣ What is "JavaScript + SWC"?

SWC (Speedy Web Compiler) is a fast JSX compiler option for better performance.

4️⃣ What is Babel?

Babel is a JavaScript compiler that converts modern ES6+ code into browser-compatible versions.

5️⃣ What does `useState` return?

It returns an array → `[current state, update function]`.

6️⃣ How does `setCount(count + 1)` work?

It updates the state and triggers a UI re-render.

7️⃣ Why does the UI update automatically?

React uses a Virtual DOM, which updates only changed parts efficiently.

🔥 Final Summary

  • ✅ DOM is a tree-structured representation of a webpage.
  • ✅ Vite is a fast alternative to CRA for React development.
  • ✅ SWC is a faster JSX compiler than Babel.
  • ✅ Babel converts modern JS into browser-compatible code.
  • ✅ `useState` returns `[currentState, updateFunction]`.
  • ✅ `setCount` updates state and re-renders the UI.
  • ✅ React optimizes rendering using the Virtual DOM.
