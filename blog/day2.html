📌 React Study Day 2 Summary

✅ Key Concepts

  • JSX Syntax Error Fix: Self-closing tags like <img /> must be closed properly.
  • Return Statement in JSX: Use parentheses () around JSX to prevent auto-inserted semicolons.
  • map() Function Parameters: map((element, index, array) => {...})
  • Importance of Key in map(): Helps React optimize rendering performance.
  • Variable Naming in map(): You can name them freely, but using plural (users) for arrays and singular (user) for elements improves readability.
  • Loading Local Images in React: Use public for static images (/images/user1.jpg) or import for assets in src.

❓ Q&A

1️⃣ Do I have to name the variable as "users"?

Q: Do I have to name my array variable as users?

A: No, you can name it anything, but using a plural form for arrays and a singular form for elements improves readability.

2️⃣ Where does the "user" variable in map() come from?

Q: Where does user in users.map((user) => ...) come from?

A: The first parameter of map() is automatically assigned each element of the array during iteration.

3️⃣ Can I name both the array and map variable "users"?

Q: Can I do users.map((users) => ...)?

A: Yes, it will work, but it’s not recommended because it reduces readability.

4️⃣ What happens if I don't provide a key in map()?

Q: What happens if I don't include key in map()?

A: It will still work, but React may not efficiently track element changes, leading to unnecessary re-renders.

5️⃣ How do I load images from the local directory?

Q: How do I display images from my project directory?

A: Use /public for static images (<img src="/images/user1.jpg" />) or import for assets inside src.

2️⃣ Create a UserCard Component

Task: Create a UserCard component that dynamically displays a user’s name and job.

Solution:

function UserCard(props) {
  return (
    <div>
      <p>👤 Name: {props.name}</p>
      <p>💼 Job: {props.job}</p>
    </div>
  );
}

3️⃣ Render Multiple UserCards

Task: Use map() to render multiple UserCard components.

Solution:

const users = [
  { id: 1, name: "Kim", job: "Software Engineer" },
  { id: 2, name: "Lee", job: "Data Scientist" },
  { id: 3, name: "Park", job: "Product Manager" }
];

function UserList() {
  return (
    <div>
      {users.map((user) => (
        <UserCard key={user.id} name={user.name} job={user.job} />
      ))}
    </div>
  );
}

Task: Add an image to UserCard and apply basic styling.

function UserCard({ name, job, image }) {
